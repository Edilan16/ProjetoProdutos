# ProjetoProdutos

Projeto em Java para simular o cadastro e controle de estoque de um produto via console.

## Funcionalidades
- Cadastrar produto (nome, preço e quantidade)
- Exibir dados do produto e valor total em estoque
- Atualizar nome e preço do produto
- Adicionar produtos ao estoque
- Remover produtos do estoque

## Conceitos utilizados
- Programação Orientada a Objetos (POO)
- Construtores
- Encapsulamento (getters/setters)
- Método `toString()`

## Como executar
1. Abra o projeto em uma IDE Java (ex.: Eclipse/IntelliJ).
2. Execute a classe `aplication.Program`.
3. Siga as instruções exibidas no console.

## Estrutura
- `src/aplication/Program.java`: classe principal (entrada do programa)
- `src/entities/Product.java`: entidade do produto e regras de estoque
